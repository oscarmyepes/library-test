# CategoryCarouselWrapper

`CategoryCarouselWrapper` fetches a list of categories from our API and displays the list of categories in a carousel.

If you want to use your own data and for more information about category carousel component, please have a look at [CategoryCarousel](/?path=/docs/components-categorycarousel--basic)
