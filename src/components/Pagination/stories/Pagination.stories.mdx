## ClassNames you can override

You can pass a className to the root element to customize styles:

<Source language="js" code={dedent`<Pagination className="my-class-name" />`} />

Also, you can override styles using specific class names:

```
// Class for arrow left icon (previous page)
.Sui-Pagination--prev-page {...}

// Class for arrow right icon (next page)
.Sui-Pagination--next-page {...}

// Class for each page number
.Sui-Pagination--page-number {...}

```
