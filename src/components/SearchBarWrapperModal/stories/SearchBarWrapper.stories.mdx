# SearchBarWrapperModal

`SearchBarWrapperModal` is a component to show the [SearchBarWrapper](?path=/docs/wrappers-searchbarwrapper--basic) in a modal.

For more details, please read the documentation for [SearchBarWrapper](?path=/docs/wrappers-searchbarwrapper--basic).

### Usage

To use this component, please add the same properties you use to configure the SearchBarWrapper plus 2 new properties to show/hide the modal

- `show`: Boolean value that indicates when the modal should be visible.
- `onClose`: Function called every time the user clicks the close icon or when clicks the modal backdrop.

```
<SearchBarWrapperModal
    {...args}
    show={show}
    onClose={() => setShow(false)}
/>
```

## Basic

<Canvas>
  <Story
    name="Basic"
    args={{
      show: false,
    }}
  >
    {(args) => {
      const [show, setShow] = useState(args.show);
      useEffect(() => {
        setShow(args.show);
      }, [args.show]);
      return (
        <>
          <button onClick={() => setShow(true)}>Show modal</button>
          <SearchBarWrapperModal
            {...args}
            show={show}
            onClose={() => setShow(false)}
          />
        </>
      );
    }}
  </Story>
</Canvas>
