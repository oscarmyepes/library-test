# BreadCrumb

`BreadCrumb` component allow users to make selections from a range of values.

The component renders the path of the current page and a list of selected filters.

Calculates automatillcay when to collapse the breadcrumb, you can take control of that passing a `maxItems` value in the `paths` object.

**Caveat:** If you change the style of the eliipsis element (`#sui-breadCrumb--ellipsis`), the math to automatically collapse the breadcrumb won't work as expected because the width of the container will change.
If you want to set a different style to the ellipsis, please make sure the HTML element for the ellipsis (`#sui-breadCrumb--ellipsis`) width is `41px`.
