# CategoryListWrapper

The `CategoryListWrapper` component renders a list of product categories, can be configured to show categories in a grid or in a list.

If you want to use your own API, please have a look at [CategoryList](?path=/docs/components-categorylist--basic#categorylist).

## ClassNames you can override

You can pass a className to the root element to customize styles:

<Source
  language="js"
  code={dedent`<CategoryList className="my-class-name" />`}
/>

Also, you can override styles using specific class names:

```
.Sui-List--container

// Class added when the list is displayed in horizontal mode
.Sui-List--horizontal

.Sui-CategoryList--root

.Sui-CategoryList--image

.Sui-CategoryList--title
```
