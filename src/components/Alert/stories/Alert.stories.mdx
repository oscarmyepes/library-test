## ClassNames you can override

You can pass a className to the root element to customize styles:

<Source language="js" code={dedent`<Alert className="my-class-name" />`} />

Also, you can override styles using specific class names:

```
// Class name for the root element
.Sui-Alert--root
// Class name for the icon on the left
.Sui-Alert--icon
// Class name for text container
.Sui-Alert--info
// Class name for the title
.Sui-Alert--title
// Class name for the text
.Sui-Alert--text
// Class name for the button container
.Sui-Alert--action

```
