# SpecificationSearch

The specification search component receives an array of tuples and shows matched results based on a search string.
The first item in the tuple is the `attribute name` and the second one is `attributes value`.

Example:

If you have a product with these attributes

<Source
  language="js"
  code={`
{
  "stockid": "AB-12345",
  "brand_name": "Sunhammer",
  "condition": "New",
  "item_number": "37050-6",
  "engine: "Toyota",
};
  `}
/>

You can transform that object to an array of tuples to something like this::

<Source
  language="js"
  code={`
[
  ["Stock ID", "AB-12345"],
  ["Brand", "Sunhammer"],
  ["Condition", "New"],
  ["Item Number", "37050-6"],
  ['Engine Make', 'Toyota'],
];
  `}
/>

When users start typing in the search bar, the component will show a list of matches based of the values in the tuple you sent to the component.
