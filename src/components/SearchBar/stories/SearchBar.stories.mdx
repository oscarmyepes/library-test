# SearchBar

SearchBar also called WSS (Web Shop Search) uses an incremental search philosophy and starts sending search requests once the user has typed in 3 or more characters into the search bar (the number of characters can be configured).

Features:

- Customers can configure what fields should appear in the result items list by sending a property **visibleFields** with the fields and labels to be displayed.
  _Example:_

```
const visibleFields = [
  { prop: 'title' },
  { prop: 'brand_name', label: 'BRAND: ' },
  { prop: 'dealerid', label: 'PRODUCT #: ' },
  { prop: 'sale' },
  { prop: 'price' },
]

```

- Text in the result list can be configured as well, Header result (like SHOWING RESULTS FOR: {search}), list title (Products) can be configured.
- Customers can navigate results using arrow keys (↑↓) and select a product by hitting the Enter key.
