# FitmentSelectorVerifier

This component is a specific use case of [FitmentSelectorWrapper](?path=/docs/wrappers-fitmentselectorwrapper--basic) and [Alert](/?path=/docs/components-alert--basic-no-styles).

Here we show how the **Alert** and **FitmentSelectorWrapper** components work together.

On first load, if the fitment does not have all selected values or if they don't match what we have in our database, an error message will be displayed,
then user can click **Change vehicle** button to select the proper fitment for the vehicle he is looking for.

## Error

When there is not enough selected data in the Fitment or if there is a wrong id in the `selectedValues` object, we show an error message.

In this example the error happens because the selected **Make** id is not our in database.
