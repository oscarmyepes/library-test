# CategoryList

The `CategoryList` component renders a list of product categories, can be configured to show categories in a grid or in a list.

When you use `<CategoryList />` component, you have to send all data. This component is useful if you don't want to use our API and you want to use you own data.

## ClassNames you can override

You can pass a className to the root element to customize styles:

<Source
  language="js"
  code={dedent`<CategoryList className="my-class-name" />`}
/>

Also, you can override styles using specific class names:

```
// Class added when the list is displayed in horizontal mode
.Sui-List--container

.Sui-CategoryList--root

.Sui-List--horizontal

.Sui-CategoryList--image

.Sui-CategoryList--title
```
